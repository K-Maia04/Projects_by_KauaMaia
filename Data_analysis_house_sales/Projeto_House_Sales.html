Projeto de Anlise de Dados e Previso de Preos das casas em King County, USA

Primeiramente, esse projeto tem como objetivo fazer uma anlise de dados mais cautelosa e profissional utilizando mtodos estatistcos para a anlise de caracteristcas que mais impactam os valores das casas de King County, alm da criao de um modelo que possa prever os valores das casas com base nas caracteristcas dadas.

Vale ressaltar, que esse projeto usa como base um Dataset disponibilizado pela IBM durante o Projeto Final do curso 'Data Analysis with Python', na plataforma Coursera.

Sobre o Dataset

Este dataset contm preos de venda das casas de King County, que inclui Seattle. Incluindo, as casas vendidas entre maio de 2014 e maio de 2015. Foi retirado aqui. Este dataset foi modificado pelos professores do curso.

Variable Description
id A notation for a house
date Date house was sold
price Price is prediction target
bedrooms Number of bedrooms
bathrooms Number of bathrooms
sqft_living Square footage of the home
sqft_lot Square footage of the lot
floors Total floors (levels) in house
waterfront House which has a view to a waterfront
view Has been viewed
condition How good the condition is overall
grade overall grade given to the housing unit, based on King County grading system
sqft_above Square footage of house apart from basement
sqft_basement Square footage of the basement
yr_built Built Year
yr_renovated Year when house was renovated
zipcode Zip code
lat Latitude coordinate
long Longitude coordinate
sqft_living15 Living room area in 2015(implies-- some renovations) This might or might not have affected the lotsize area
sqft_lot15 LotSize area in 2015(implies-- some renovations)

Agora, iremos treinar o modelo utilizando a Regresso Linear Mltipla, e por meio do mtodo da Cross-Validation ser possvel checar se nosso modelo est OverFitting ou UnderFitting nossos dados.

Para facilitar e padronizar o treino, foi criado um Pipeline para facilitar o processo. Irei testar polinmios de ordem 1,2,3 que so os mais comuns para estimativas de dados.

In[]:
# Definindo os graus dos polinmios
order = [1, 2, 3]

# Inicializando listas vazias para uso da funo
Rsqu_test = []
Rsqu_train = [] 
Mean_List = {}  
Std_List = {}

# Criando um Pipeline com PolynomialFeatures e LinearRegression
for n in order:
    Input = [('scale',StandardScaler()),('polynomial', PolynomialFeatures(degree=n,include_bias=False)),('model',LinearRegression())]
    pipeline = Pipeline(Input)
    
    # Ajustando o pipeline para a data de treino
    pipeline.fit(x_train[features], y_train)
    
    # Avaliando o modelo usando Cross_Validation
    cv_scores = cross_val_score(pipeline, x_train[features], y_train, cv=5)
    mean_r2 = np.mean(cv_scores)
    Mean_List[f'Grau {n}'] = mean_r2.round(3)
    Std_List[f'Grau {n}'] = cv_scores.std().round(4)

    # Calculando o Rsquared da data teste e treino
    Rsqu_test.append(pipeline.score(x_test[features], y_test))
    Rsqu_train.append(pipeline.score(x_train[features], y_train))

    
# Criando um DataFrame dos resultados para melhor visualizao
nwdf = pd.DataFrame([Mean_List, Std_List])
nwdf.rename(index={0: 'Mdia de Rsquared', 1: 'Desvio Padro'}, inplace=True)
display(nwdf)

# Plotando os resultados para melhor visualizao
plt.figure(figsize=(10, 5))
